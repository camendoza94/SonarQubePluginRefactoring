DTOs must have a constructor with an Entity as parameter.

It must check if the entity object is not null and then copy its values to the DTO

Noncompliant Code Example

public EditorialDTO(EditorialEntity editorial) {
    this.id = editorial.getId();
    this.name = editorial.getName();
}

Compliant Solution

public EditorialDTO(EditorialEntity editorial) {
    if(editorial != null) {
        this.id = editorial.getId();
        this.name = editorial.getName();
    }
}
